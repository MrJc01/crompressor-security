# Websocket Chat Stream

Testa a resiliência do Cérebro P2P contra conexões contínuas e abertas no formato Stream de WebSockets. 
O Proxy engole a stream e consegue comutar a persistência do Túnel TCP no Swarm Node P2P.
O Mock utiliza chamadas diretas via UDP Socket Bash para provar que a conexão não é fechada automaticamente pelo TimeOut.
