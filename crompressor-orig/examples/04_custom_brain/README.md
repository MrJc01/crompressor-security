# 04 - O Cérebro Especializado (Codebooks Limitados)

A mágica LSH do CROM permite treinar Codebooks (o cérebro do compressor) focado apenas numa bolha de entropia — por exemplo, log de banco de dados, bases de DNA ou textuais curtos.

Um Codebook Especializado em Código Go não conseguirá comprimir um arquivo .mp4. Esta propriedade garante uma **camada de Segurança Baseada em Contexto** (*Obscuridade Contextual*).

## Execute
```bash
./run.sh
```
O script treinará o CROM para ser um Gênio apenas na compreensão de arquivos Texto. Depois, tentaremos usá-lo para comprimir um JPEG. O ratio será nulo.
